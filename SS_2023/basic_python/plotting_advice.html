Make your plots sexy again!

Size and Resolution

The size of the figure is given by:

fig = plt.figure(figsize=(6,4))
  • Use small numbers! Do not make a large figure and rearrange the label size, legend size,... Instead:

The resolution of the figure is set by dpi:

fig.savefig("name.png", dpi=256)

Legibility of a Plot

Use a grid:

plt.grid(linestyle=(0,(5, 10)), linewidth=0.5)

Include ticks on all axes:

import matplotlib
matplotlib.rcParams['xtick.direction'] = 'in'
matplotlib.rcParams['ytick.direction'] = 'in'
matplotlib.rcParams['xtick.top'] = True
matplotlib.rcParams['ytick.right'] = True

If you have words on one axis you can rotate the tick labels:

plt.xticks(rotation=90)

Use logarithmic scale and do not label, e.g., log(E):

plt.xscale("log")

Use LaTex for axes labeling and legend:

plt.xlabel(r"rate of \nu_\mu / \frac{1}{s}")

You can use formatters on numbers on your axes. This is especially useful for degrees:

import matplotlib.ticker as ticker
plt.gca().xaxis.set_major_formatter(ticker.FormatStrFormatter(u'%dÂ°'))

Units are not given in brackets (energy E [eV]):

  • brackets are usually used as operator for "unit of" (according to ISO, NIST, IUPAC, IUPAP...) 1 2
  • write "energy E / eV" or "energy E in eV"

If the figure is cut off after saving, use:

plt.tight_layout()

Be nice to colorblind people! Most important, do not use red-green plots of the same brightness. You can check online for colorblind color palettes and use e.g. different linestyles and markers.

Colorbar

List of colormaps

  • Never use jet!
  • For a symmetric colorbar around zero or a certain value use a diverging colormap such as seismic. Make sure that it is centered around zero!
  • For non-symmetric colorbars use sequential colormaps, ideally colormaps that are still readable in gray-scale (e.g. cubehelix)
  • Label your colorbar
  • There are some colorbars for discrete steps, but you can customize a colorbar by your own:
color = array(["w", "silver", "dimgrey", "k", "yellow", "gold", "darkorange", "red", "firebrick", "lime","mediumseagreen", "darkgreen", "lightcyan", "cyan", "dodgerblue", "navy"])
cmap = colors.ListedColormap(color)
plt.pcolormesh(x, y, matrix, cmap=cmap, vmin=0, vmax=int(matrix.max())+1)
